| T-04-11 | Denial of Service | A hung dependency stalling the health handler | mitigate | Each probe runs under a shared 3s `context.WithTimeout` (D-16); a stalled Postgres/Redis/S3 returns 503 quickly rather than hanging the handler. |
| T-04-12 | Tampering | Health probe mutating dependency state | mitigate | Probes are read-only: `pgxpool.Ping`, redis `PING`, MinIO `BucketExists` — no INSERT, no object PUT/GET (D-16). |
