<objective>
Add fail-closed magic-bytes detection for the five video container formats (mp4, mov, avi, mkv, webm) — the AVE-01 half of Phase 34 — as a standalone, unit-tested addition to the existing `internal/convert` sniffer surface. mp4/mov/avi fit the existing fixed-12-byte-window mechanism and extend `sniff.go`'s `signatures` table directly; mkv/webm share an identical EBML magic and require a genuinely new bounded-peek EBML/DocType walker (the single highest-uncertainty item in this phase).

Purpose: A misdetected or spoofed container must fail closed BEFORE any byte reaches ffmpeg or S3 (V5/V8 input validation). This is built and tested this phase but consumed by the API only in a later phase.

Output: `internal/convert/avsniff.go`, `internal/convert/avsniff_test.go`, and modifications to `internal/convert/sniff.go`.
</objective>

|-----------|----------|-----------|-------------|-----------------|
| T-34-01 | Spoofing | matchMP4/matchMOV/matchAVI/matchEBML | mitigate | Fail-closed magic-bytes matching keyed on brand/DocType, not extension; a spoofed extension cannot pass detection |
| T-34-02 | Tampering | Cross-sniffer collision (video misdetected as audio/image, or vice versa) | mitigate | TestVideoBrandDisjointness asserts pairwise-empty intersection of mp4VideoBrands/m4aBrands/heicBrands, making the disjointness a permanent, cross-file invariant |
| T-34-03 | Denial of Service | matchEBML declared-length parsing | mitigate | Bounded 4KiB peek (avPeekLen); every declared vint size/offset that would run past the buffer returns false — never grows the buffer, never seeks the stream |
| T-34-04 | Tampering | Malformed/truncated EBML header | mitigate | matchEBML returns ("", false) on truncation or unrecognized DocType — never guesses mkv/webm, so a crafted partial header cannot be laundered into a valid classification |
| T-34-SC | Tampering | npm/pip/cargo installs | accept | No package-manager dependencies added this phase (ffmpeg/ffprobe are OS binaries, no matcher uses them); RESEARCH.md Package Legitimacy Audit is not-applicable |
