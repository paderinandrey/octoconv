| Boundary | Description |
|----------|-------------|
| client→API (multipart) | client-supplied `preset` name form field crosses into a DB lookup; must be opaque, length-bound, and non-leaking on miss |
| presets table→converter | stored preset options are operator-originated durable state re-validated at every job creation |
| resolve→insert window | a preset can be deactivated/bumped between resolution and the job-row insert (TOCTOU) |
