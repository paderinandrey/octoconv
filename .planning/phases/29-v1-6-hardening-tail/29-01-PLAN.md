| helm upgrade → running Prometheus pod | A ConfigMap change that never rolls the pod leaves a stale scraper feeding KEDA |

| T-29-01 | Denial of Service | scaledobject-*.yaml trigger | mitigate | `ignoreNullValues: "false"` (D-01) converts sustained metric absence to a scaler error so `fallback.replicas: 1` holds a busy class up instead of false scale-to-zero during an api outage |

| T-29-04 | Tampering | falsy-0 stabilization | mitigate | `hasKey`/nil-guard (D-06 fix #1) preserves an operator's explicit `stabilizationWindowSeconds: 0` instead of silently applying the 300s default |
