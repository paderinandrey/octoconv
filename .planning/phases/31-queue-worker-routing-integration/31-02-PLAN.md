## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| downloaded audio file → EnforceMaxDuration → ffprobe | untrusted downloaded bytes measured before decode |
| downloaded audio file → ffmpeg -i path arg | path handed to a tool that interprets URL/protocol specifiers |
| engine error string → terminal/transient classifier | error message drives retry vs fail-closed decision |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-31-01 | Denial of Service | duration guard built but never invoked (T-30-08 carry-forward) | mitigate | EnforceMaxDuration spliced into process() after download, before Convert, gated on EngineAudio; ErrAudioDurationExceeded classifies terminal; IN-02 pinning test proves the guard runs before ffmpeg. |
| T-31-03 | Tampering / SSRF-adjacent | ffmpeg/ffprobe treating path arg as URL/protocol specifier (IN-01) | mitigate | file: protocol prefix on the ffprobe path arg and the ffmpeg -i arg — prevents concat:/http:/pipe:/leading-dash reinterpretation of a future client-influenced filename. |
| T-31-07 | Denial of Service | whisper-stage timeout retried unboundedly | mitigate | isAudioTerminal keeps whisper-stage timeout transient but retries are bounded by AUDIO_MAX_RETRY (asynq MaxRetry); ffmpeg-stage failure is terminal (no retry on malformed input). |
