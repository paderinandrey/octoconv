| Boundary | Description |
|----------|-------------|
| upstream ffmpeg git tag → build stage | A force-moved tag could inject malicious source into the compiled binary |
| ffmpeg decoder ← attacker-supplied video (runtime) | Untrusted container bytes decoded by the compiled ffmpeg; CVE-class decoder RCE surface |
| ffmpeg protocol layer ← file content refs | HLS/concat/subtitle references embedded in uploads could reach network protocols |
| container process ← host | av-worker runs untrusted-input engine; privilege must be minimal |
