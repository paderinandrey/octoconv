## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| KEDA/HPA → av-worker Deployment | The autoscaler owns the 0↔N replica decision; a wrong grace/stabilization value here lets it SIGTERM a live ffmpeg transcode |
| Prometheus → ScaledObject PromQL | An absent/empty metric series (api down, unscrapeable, or a retry-only backlog) crosses here and, mishandled, scales a busy class to zero |
| Chart ConfigMap vs docker-compose env | Two independently-maintained env surfaces (AV_*) that can silently drift |
